# 🧠 AI-Powered Fake News Detection System

## 📌 Overview
This project is a Machine Learning-based system that detects whether a news article is Fake or Real using Natural Language Processing techniques.

## 🚀 Features
- Fake vs Real news classification
- Confidence score for predictions
- Important word highlighting for explainability
- Interactive UI using Streamlit

## 🧠 Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF)
- Streamlit

## 📊 Model
- Logistic Regression
- Accuracy: ~95%

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
